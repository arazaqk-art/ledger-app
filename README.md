This is a Python-based Ledger App Professional, a customer ledger management system developed by Abdul Razzaque. It is built using FastAPI, a high-performance web framework for building APIs, ensuring fast performance and easy maintenance. 
🛠️ Key Features
📊 Dashboard Totals: Real-time, at-a-glance overview of your financial metrics.
🔍 Customer Search: Quickly find specific customer records and transaction histories.
📄 PDF Invoice: Generate professional, downloadable PDF invoices for transactions.
📈 Excel Export: Export data for accounting and analysis in spreadsheet format.
👥 Staff Management: Manage user access and staff roles.
🔐 Login System: Secure access to the dashboard with built-in authentication. 
🚀 Run Locally 
To run this application on your local machine, follow these steps:
Install Requirements:
bash
pip install -r requirements.txt
Run the Application:
bash
uvicorn main:app --reload
Access the Application:
Open your browser and navigate to: http://127.0.0.1:8000
